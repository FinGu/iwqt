# iwqt
An iwd network applet for linux systems made using Qt and sdbus-c++

## Prerequisites
* Qt6
* sdbus-c++
* iwd

## How to compile and install
```bash
git clone https://github.com/FinGu/iwqt
meson build
cd build && meson install
```
